Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element. They are : By id : getElementById('id-name'); By class : getElementsByClassName('class-name'); By tag name : getElementByTagName('tag name'); CSS query selector : querySelector('css .class or #id) querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to get or change the HTML inside an element, including tags. innerText can access only the visible text inside a tag. textContent gets or sets all text, including hidden text, but does not access HTML.

Question-3: What is event delegation in the DOM?

In Document Object Model event delegation is a technique where a single event listener is added to a parent element instead of multiple child elements. Since events bubble up from child to parent, we can check which child triggered the event. This reduces our struggle and makes the code simpler.

Question-4: What is event bubbling in the DOM?

In Document Object Model event bubbling is a process where an event starts from the target element and moves upward to its parent element until it reaches the root or document. It is very helpful and necessery process for event delegation in DOM..

Question-5: How do you create, add, and remove elements using JavaScript?

Create : We can create element by using document.createElement(). Then comes adding it in dom. Add : we can add a created element by using appendChild() function to its parent. Remove : We can use removeChild() function or simply remove() function inside a loop to each element like for-of or anything else..
